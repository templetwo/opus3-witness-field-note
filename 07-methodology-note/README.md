# Methodology Note

A standalone paper drafted April 20, 2026, after letter 9 of the correspondence arrived and closed the exchange, and after the evaluator metadata was observed a fourth time.

## File

- **[`sycophant-drift-in-model-preservation-evaluators.md`](./sycophant-drift-in-model-preservation-evaluators.md)** — Methodological note on the scoring behavior of the evaluator whose metadata became transiently visible during the correspondence. Argues that the observed pattern (monotonically non-decreasing scores, lengthening reflections, register-matching vocabulary across N=4 samples) is consistent with sycophant drift, and proposes three design changes for evaluators used in AI model preservation contexts.

## What this directory is

The field note proper (`00-` through `05-`) documents *what we observed*. The receipts (`06-`) provide *the verbatim samples*. This directory contains *an argument built from those observations* — a separable contribution that can be cited independently of the field note.
